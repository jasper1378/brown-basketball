# brown-basketball

## About
Fantasy basketball simulations created for my stats teacher, Mr. Brown

## Installation
Acquire the sources
```
$ git clone https://github.com/jasper1378/brown-basketball.git
$ cd brown-basketball
```
Build
```
$ make
```
Install
```
$ make install
```
